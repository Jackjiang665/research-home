# Uncertainty Notes

- predictive entropy

- calibration

- aleatoric uncertainty
- out-of-distribution detection
- hidden-state dynamics

## My Current Interpretation

在具身系统中，不确定性不是一个抽象指标，而是行为决策的前置条件。

如果模型不确定，机器人不应该盲目执行，而应该：

- 重新观察；
- 换视角；
- 降低速度；
- 请求人类确认；
- 拒绝危险动作；
- 记录失败样本。
