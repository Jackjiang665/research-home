# Embodied AI Notes

## My Interest

我更关心真实机器人系统中的智能，而不是只在离线 benchmark 上刷分。

核心问题：

> 机器人如何在真实环境中理解任务、判断风险、执行动作，并在失败或不确定时恢复？

## Topics

- robot manipulation
- robot navigation
- vision-language-action models
- imitation learning
- active perception
- task planning
- skill library
- failure detection
- recovery strategy

## Personal Background

本科机器人队经历让我熟悉无人机、小车、搬运、农业植保、轮式机器人等工程项目。未来希望把这些真实系统经验和多模态模型、具身智能结合起来。
