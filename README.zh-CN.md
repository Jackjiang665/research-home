# 姜函辰

[English](README.md)

北京邮电大学 2026 级准研究生，本科专业为电气工程及其自动化。长期参与机器人竞赛和工程项目，关注真实机器人系统中的多模态感知、决策可靠性和自适应恢复。

我希望研究的不是只在离线数据上跑分的模型，而是能进入真实机器人任务流程的算法：机器人能看懂任务、判断自己的感知和决策是否可靠，并在不确定或失败时采取补救动作。

## 当前关注

我目前以 **VLM 输出不确定性估计** 作为进入具身智能可靠性方向的切入点。

当前研究问题：

> 基于 hidden-state dynamics 的方法，能否检测和归因 VLM 在视觉扰动下的输出不确定性，并进一步成为机器人执行前的可靠性判断模块？

## 研究兴趣

| 方向 | 我关心的问题 |
|---|---|
| 具身智能系统 | 如何把感知、规划、技能库和控制接到真实机器人任务中 |
| 机器人自我诊断 | 机器人如何知道自己什么时候可能失败、为什么失败 |
| 自适应恢复 | 不确定时如何重拍、换视角、请求确认或重新规划 |
| 多模态可靠性 | 不只看 benchmark accuracy，而是评估 VLM/VLA 输出能否支撑真实行动 |
| 模型部署可靠性 | 压缩、量化、端侧部署后模型可靠性如何变化 |
